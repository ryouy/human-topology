# 除外ルール

**最優先**: 「人物だけのリンク距離空間」を壊さないこと。一覧・曖昧さ回避などを経由点に含めない。

## 1. ノードとして含めないもの

以下に該当するページは**ノードにしない**。

- **一覧ページ**（タイトルまたはカテゴリに「一覧」が強く関連するもの）
- **曖昧さ回避**（pageprop `disambiguation`、またはカテゴリ「曖昧さ回避」）
- **カテゴリページ**（名前空間が Category）
- **テンプレート・ヘルプ・Wikipedia・ファイル・ポータル** などメイン記事空間外
- **年号ページ**（タイトルが「○○年」のみ、などヒューリスティック）
- **月日ページ**（「○月○日」形式など）
- **作品・組織・地名・事件** に分類される記事（カテゴリ・タイトルパターンで除外）

## 2. 経路としても使わない

グラフのエッジは「人物 → 人物」のみ。非人物へのリンクは**辺として張らない**。したがって BFS の距離は自動的に人物のみになる。

## 3. 実装での判定（MVP）

### 3.1 強制除外

- `pageprops` に曖昧さ回避用フラグがある
- タイトルが `一覧` で終わる、または `の一覧` を含む（例外的に人物名が「一覧」で終わるケースは稀として除外優先）
- 名前空間がメイン（0）以外（API では通常メイン記事のみ取得）

### 3.2 人物であることの肯定

次のいずれかに近いことを満たす（カテゴリ・要約のキーワード）。

- 「人物」「人物」に関するカテゴリ
- 生年・没年・「生没年」に関するカテゴリ
- 「日本の」「日本人」＋職業・分野を示すカテゴリ（政治家、俳優、作家 等）

### 3.3 日本人であることの肯定

次のいずれか。

- カテゴリに `日本` / `日本人` を含むものがある（ただし「日本の映画」だけでは人物と限らないため、人物判定と組み合わせる）
- 要約に「日本の」「日本人」＋職業表現がある（補助）

### 3.4 非人物のヒューリスティック

カテゴリまたはタイトルに以下が**主題**として付く場合は除外。

- `作品`, `映画`, `小説`, `アルバム`, `楽曲`, `組織`, `会社`, `大学`, `地名`, `事件`, `戦争`

（完全一致ではなく部分一致。誤爆を減らすため、MVP では「カテゴリ名に ○○の作品 などが含まれる」場合を優先）

## 4. ダンプ版への拡張

`page_props` の `hiddencat` / `disambiguation`、`categorylinks` による網羅的な分類へ移行する際も、**同じ意味の除外**を維持する。
